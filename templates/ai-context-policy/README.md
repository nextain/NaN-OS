# AI Context Policy Template

**Naia OS에서 추출한 AI 오픈소스 정책 템플릿**

다른 오픈소스 프로젝트에서 AI 컨텍스트 구조, 라이선스, 기여 가이드, 도네이션 정책을 빠르게 적용할 수 있도록 만든 템플릿입니다.

## 포함 파일

| 파일 | 설명 |
|------|------|
| `AGENTS.md.template` | AI 에이전트 진입점 (프로젝트 규칙 요약) |
| `agents-rules.json.template` | 프로젝트 핵심 규칙 SoT |
| `philosophy.yaml.template` | 프로젝트 철학 (왜 만드는가) |
| `contributing.yaml.template` | AI/인간 기여 가이드 |
| `donation.yaml.template` | 도네이션 정책 |
| `project-index.yaml.template` | 컨텍스트 인덱스 + 미러링 규칙 |

## 사용법

1. 이 디렉토리를 복사
2. `{{PROJECT_NAME}}`, `{{ORG_NAME}}`, `{{YEAR}}` 등 플레이스홀더 교체
3. `.agents/context/`에 배치
4. `.users/context/` 미러 (한국어/영문) 생성
